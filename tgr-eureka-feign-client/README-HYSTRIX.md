在分布式系统中，服务与服务之间的依赖错综复杂， 一种不可避免的情况就是某些服务会
出现故障，导致依赖于它们的其他服务出现远程调度的线程阻塞 。 Hystrix 是 Netflix 公司开
源的一个项目，它提供了熔断器功能，能够阻止分布式系统中出现联动故障 。 Hystrix 是通过
隔离服务的访问点阻止联动故障的，并提供了故障的解决方案，从而提高 了整个分布式系统
的弹性 。
